### 倡导和平有爱共处，反对建政开盒互怼

### 安装
1. Clone 仓库
2. 安装 Node.js
3. 安装 Yarn：`npm i -g yarn`
4. 安装库文件(在项目文件夹内)：`yarn`

AI/OpenCV:

1. cd ./ai
2. pip install -r requirements.txt

视频抽帧:

1. 安装 FFmpeg 并放入 Path 中

### 配置
1. 登录 SelfBot：打开命令行，输入 `yarn start`，按提示登录

### 运行（保活）

打开命令行，输入 `yarn run keep`

AI/OpenCV: 
1. cd ./ai
2. python api.py
